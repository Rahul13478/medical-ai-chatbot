# Medical AI Chatbot

A conversational AI chatbot built with Python and Google Gemini API.
The chatbot is specialized to only answer medical related questions.

## Technologies Used
- Python
- Google Gemini API
- FastAPI (coming soon)

## How to Run
1. Clone the repository
2. Install dependencies: pip install google-genai python-dotenv
3. Create a .env file and add your GEMINI_API_KEY
4. Run: python ai.py
